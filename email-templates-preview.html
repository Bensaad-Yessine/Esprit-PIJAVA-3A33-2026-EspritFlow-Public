Bonjour Marie,

Nous avons recu une demande de reinitialisation de mot de passe pour votre compte EspritFlow.

Code de reinitialisation
abc123.def456ghi789jkl

[TEMPS LIMITE] Attention : Ce code expire a 15:30 (dans 30 minutes).

Etapes :

  1. Ouvrez l'application EspritFlow
  2. Cliquez sur "Forgot password?" sur l'ecran de connexion
  3. Entrez votre adresse e-mail
  4. Entrez le code ci-dessus
  5. Creez un nouveau mot de passe securise
  6. Cliquez sur "Reinitialiser le mot de passe"

[SECURITE] Important : Si vous n'avez pas demande cette reinitialisation, ignorez cet email. Votre compte restera securise.
