COMMENT COMMENCER (3 ÉTAPES)
═════════════════════════════════════════════════════════════════════════════════

ÉTAPE 1: Configurer SMTP
──────────────────────────
Éditez le fichier: C:\Users\MSI\IdeaProjects\pijava\.env

Ajoutez/mettez à jour:
   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_USER=votre-email@gmail.com
   SMTP_PASSWORD=votre-mot-de-passe-app
   SMTP_FROM=votre-email@gmail.com
   SMTP_STARTTLS=true
   SMTP_SSL=false

Pour Gmail:
   1. Allez à: https://myaccount.google.com/apppasswords
   2. Sélectionnez: Mail + Votre Appareil
   3. Générez un mot de passe (16 caractères)
   4. Copiez dans .env (sans espaces)

ÉTAPE 2: Tester la Réinitialisation de Mot de Passe

DÉPANNAGE RAPIDE
═════════════════════════════════════════════════════════════════════════════════

PROBLÈME: Email non envoyé
SOLUTION:
   ├─ Vérifiez les crédentials SMTP dans .env
   ├─ Vérifiez que le serveur SMTP est accessible
   └─ Vérifiez les logs pour les messages d'erreur

PROBLÈME: Gmail rejette l'email
SOLUTION:
   ├─ Utilisez le mot de passe d'application, pas le mot de passe du compte
   └─ Générez-le depuis: https://myaccount.google.com/apppasswords

PROBLÈME: Les caractères français s'affichent mal
SOLUTION:
   └─ UTF-8 est activé par défaut, vérifiez la configuration du client email

PROBLÈME: Cannot compile (erreur de compilation)
SOLUTION:
   └─ Vérifiez que toutes les dépendances sont disponibles
