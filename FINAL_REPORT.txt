┌─ INTEGRATION POINTS ─────────────────────────────────────────────────────────────┐
│                                                                                   │
│  Password Reset Flow:                                                            │
│  1. User clicks "Forgot password?" (loginController.java)                       │
│  2. Opens ForgotPasswordController                                               │
│  3. User enters email                                                            │
│  4. PasswordResetService.requestReset() called                                   │
│  5. HTML email generated and sent via MailService                                │
│  6. User enters code in ResetPasswordController                                  │
│  7. PasswordResetService.resetPassword() validates and updates                   │
│                                                                                   │
│  Ban Flow:                                                                       │
│  1. Admin calls UserServices.banUser(userId, reason)                             │
│  2. BanNotificationService.notifyUserBanned() called                             │
│  3. HTML email generated and sent via MailService                                │
│  4. User receives ban notification                                               │
│                                                                                   │
│  Unban Flow:                                                                     │
│  1. Admin calls UserServices.unbanUser(userId)                                   │
│  2. sendUnbanNotificationEmail() called                                          │
│  3. HTML email generated and sent via MailService                                │
│  4. User receives success notification                                           │
│                                                                                   │
