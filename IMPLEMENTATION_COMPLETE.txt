INTEGRATION POINTS
════════════════════════════════════════════════════════════════════════════════

1. Password Reset Flow
   ├─ Triggered by: User clicks "Forgot password?"
   ├─ Location: ForgotPasswordController.java
   ├─ Service: PasswordResetService.requestReset()
   ├─ Email: Automatically sent with reset code
   └─ Duration: Token valid for 30 minutes

2. Ban Notification Flow
   ├─ Triggered by: Admin bans user
   ├─ Location: UserServices.banUser()
   ├─ Service: BanNotificationService.notifyUserBanned()
   ├─ Email: Automatically sent to user
   └─ Content: Ban reason + appeal instructions

3. Unban Notification Flow
   ├─ Triggered by: Admin unbans user
   ├─ Location: UserServices.unbanUser()
   ├─ Service: sendUnbanNotificationEmail()
   ├─ Email: Automatically sent to user
   └─ Content: Confirmation + login instructions
