 🎯 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1. Run SQL migration:
   mysql -u root -p pidev < migration_smart_courses.sql

2. Get Udemy API key:
   • Go to: https://rapidapi.com/tutorialindex-tutorialindex-default/api/udemy-paid-courses-for-free-api
   • Subscribe to free tier
   • Copy your X-RapidAPI-Key

3. Update .env:
   UDEMY_API_KEY=paste_your_key_here

4. Update user.java entity:
   Add: currentStreak, longestStreak, lastStreakDate, totalCoursesCompleted fields

5. Add navigation to sidebar:
   Add button in FrontSidebarController → loadView("/frontoffice/courses/courses-content.fxml")

6. Run the app:
   mvn clean javafx:run

7. Test:
   Click "📚 Courses" in sidebar → Search "Python" → Enroll → Complete course
